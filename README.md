# EDA Felicidad Mundial 2024

## Descripción del proyecto
Este proyecto desarrolla un Análisis Exploratorio de Datos (EDA) sobre la felicidad mundial en 2024, con el objetivo de identificar qué factores están más asociados al nivel de felicidad de los países.

## Objetivo
Analizar la relación entre la puntuación de felicidad y diferentes variables explicativas, como el PIB per cápita, el apoyo social, la esperanza de vida saludable y la libertad para tomar decisiones vitales.

## Dataset
El análisis se ha realizado a partir de un dataset internacional sobre felicidad, filtrando únicamente los registros correspondientes al año 2024 para centrar el estudio en la información más reciente disponible.

## Principales hallazgos
- El apoyo social es la variable más fuertemente relacionada con la felicidad.
- El PIB per cápita, la salud y la libertad también muestran asociaciones positivas relevantes.
- Los países más felices no destacan por una sola variable, sino por combinar varios factores favorables al mismo tiempo.

## Estructura del proyecto
- `notebooks/eda_happiness.ipynb`: notebook principal del análisis
- `src/data/happiness.csv`: dataset utilizado
